# Motion Detector Through Camera

## Aim
Detect human posture and motion in real time using a webcam.

## Features
- Standing
- Sitting
- Walking
- Running

## Technologies
Python, OpenCV, MediaPipe, NumPy

## Installation
```bash
pip install -r requirements.txt
python motion_detector.py
```

## Project Structure
